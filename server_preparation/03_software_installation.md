# Software Installation on Azure VM

After initial OS configuration and security setup, install the necessary software. Run these commands as your sudo-privileged user.

## 1. Install Git

Git is required for cloning repositories and for the server-side bare repository that will receive pushes for deployment.
**[Git - Installation](https://git-scm.com/downloads)**

## 2. Install Nginx Web Server

Nginx will serve your static Hugo site.
**[Nginx - Installatio](https://nginx.org/en/docs/install.html)**

## 3. Install Hugo (Extended Version Recommended)

Hugo will build your static site on the server. The "extended" version is recommended as many themes require its SASS/SCSS processing capabilities.

**[Hugo - Installationx](https://gohugo.io/installation/)**
